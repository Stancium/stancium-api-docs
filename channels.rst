Channels
========
Channels are Ayaka's one-way communication type for both messages and voice.


Category
--------

+---------------+-------------------+
| name          | type              |
+---------------+-------------------+
| id            | int               |
+---------------+-------------------+
| name          | string            |
+---------------+-------------------+
| description   | string            |
+---------------+-------------------+
| inside_of     | int (always 0)    |
+---------------+-------------------+
| type          | int (always 1)    |
+---------------+-------------------+
| position      | int               |
+---------------+-------------------+
